- （イ）番方編成による交替制によることが就業規則等により定められており、制度として運用されていること。

3　本条2項においてなお定めている、いわゆる「振替休日」とは、例えば業務の都合によって所定休日である日を労働日とする必要が生じた場合に、当該所定休日を他の勤務日に変更し、その代わりの休日として別に休日を取得するもので、所定の休日と所定の勤務日とを振り替えることをいいます。

また、「代休」とは、休日に休日労働を行わせた場合に、その代わりに近接の勤務日又は労働者の希望する任意の勤務日の勤務義務を免除し、休みを与える制度であるといえます。振替休日と代休の労基法上の取扱いの違いは次のとおりです。

| 労働基準法上の振替休日と代休の取扱いの違い |
|---|
| ①　振替休日は、あらかじめ定められた法定休日を他の日に振り替えることであるから、振替前の休日に勤務しても通常の勤務と同じである。しかしながら、休日を振り替えたことにより、その週の労働時間が1週間の法定労働時間を超えるときは、その超えた時間については割増賃金の問題は発生しませんが、振り替えた休日が翌週にまたがる場合、振替勤務のことにより当該週の実労働時間が法定労働時間内の場合については当然割増賃金を支払う必要があります。 |
| ②　一方、代休は、定められた法定休日に労働させた後に、その代わりに他の日を空勤させたことを意味しているにすぎないのであるから、休日労働に対する割増賃金の問題が発生します。 |
| 休日の労働者の勤務割表がないけでそうでは、これを振り替える場合には、以下の点に留意が必要となります。 |
| ア　就業規則に振替休日の規程を置くこと。 |
| イ　振替休日は特定すること。 |
| ウ　振替休日は4週4日の休日が確保される範囲のできるだけ近接した日とすること。 |
| エ　振替は前日までに通知すること。 |
