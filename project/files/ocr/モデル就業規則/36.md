と定められています（労基則第１６条）。

**【時間外労働に関する延長時間の限度時間】**

|  | 一般の労働者の場合 | 対象期間が3か月を超える1年単位の変形労働時間制を適用する労働者 |
|---|---|---|
| 期　間 | 限　度　時　間 | 限　度　時　間 |
| 1週間 | 15時間 | 14時間 |
| 2週間 | 27時間 | 25時間 |
| 4週間 | 43時間 | 40時間 |
| 1か月 | 45時間 | 42時間 |
| 2か月 | 81時間 | 75時間 |
| 3か月 | 120時間 | 110時間 |
| 1年間 | 360時間 | 320時間 |

ただし、上記の限度時間を超えて、臨時的に時間外労働を行わなければならない特別の事情が予想される場合、特別条項付き三六協定を結ぶことにより、限度時間を超えて時間外労働時間を延長することができます。この特別条項付き三六協定は以下の要件を満たすことが必要です。

（ア）原則としての延長時間（限度時間以内の時間）を定めること。

（イ）限度時間を超えて時間外労働を行わなければならない特別の事情をできるだけ具体的に定めること。

（ウ）一定期間途中で特別の事情が生じ、原則としての延長時間を超える時間外労働を行う場合に、労使がとる手続を具体的に定めること。

（エ）限度時間を超えることのできる回数を定めること。（１年の半分以下）

（オ）限度時間を超えて延長する場合の上限の時間を定めること。また、できるだけ限り短くするよう努めること。

（カ）限度時間を超える時間外労働に係る割増賃金の率を定めること。この率は法定
